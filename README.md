# 🌐 Navegue

Um navegador web leve e rápido, focado em modularidade e eficiência. Realiza buscas integradas, exibe as principais notícias, vídeos e a previsão do clima.

## Funcionalidades
* **Busca Web:** Integração limpa com Wikipedia e Google.
* **Notícias:** Feed das principais notícias (Google News) adaptado para o idioma local.
* **Vídeos:** Atalho direto para buscas no YouTube.
* **Clima:** Previsão do tempo local em tempo real.
* **Instalável (PWA):** Pode ser adicionado à tela inicial do celular como um aplicativo nativo.

## Tecnologias
* HTML5, CSS3, JavaScript (Vanilla)
* Service Workers e Web Manifest para PWA
